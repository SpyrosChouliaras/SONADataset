# SONADataset

Resource usage and application metrics have been collected over various TPC-C executions in cloud environments. SONA Dataset includes informations about the following:

- Number of CPU cores.
- Memory size.
- Disk size.
- Disk type.
- InnoDB buffer pool size of MySQL.
- TPC-C number of warehouses.
- TPC-C number of connections.
- TpmC score.
- Average idle CPU resources in the load stage.
- Average idle CPU resources in the run stage.
- Cloud deployment monthly cost.
- Cost-performance ratio.
- System size.
- Cloud platform.
- Cloud region.
- Cloud zone.
